✅ Extrae ofertas hipotecarias reales de foros españoles

✅ Filtra contenido relevante con keywords y fechas
✅ Normaliza y exporta datos a CSV/Excel
✅ Incluye análisis estadístico de resultados
✅ Modular, configurable y documentado

================================================================================
⚠️ NOTA IMPORTANTE: PRESUPUESTO DE API
================================================================================

Durante el test detectamos que el presupuesto de EMERGENT_LLM_KEY 
está excedido. Esto es NORMAL y tiene 2 soluciones:

SOLUCIÓN 1 (RECOMENDADA - MÁS FÁCIL):
  • Ir a: Profile → Universal Key → Add Balance
  • Agregar $5 USD (suficiente para ~1000 llamadas)
  • El scraper funcionará inmediatamente sin cambios

SOLUCIÓN 2 (ALTERNATIVA):
  • Usar tu propia API Key de OpenAI
  • Editar .env: agregar OPENAI_API_KEY=tu-key
  • Ejecutar: python scraper_openai_direct.py

Ver detalles completos en: NOTA_API_KEY.txt

🏗️ ARQUITECTURA DEL SISTEMA
================================================================================

Clase Principal: MortgageScraper
├── __init__()                           # Inicialización y config
├── scrape_rankia()                      # Extrae URLs de Rankia
├── scrape_bogleheads()                  # Extrae URLs de Bogleheads
├── scrape_forocoches()                  # Extrae URLs de Forocoches
├── scrape_thread(url)                   # Procesa hilo individual
├── clean_text(html)                     # Limpia HTML → texto
├── extract_mortgage_data_with_ai()      # Extracción con GPT-4o
├── normalize_data(ofertas)              # Normaliza formato
├── save_results(df)                     # Exporta CSV/Excel
└── run()                                # Flujo principal

Flujo de Ejecución:
1. Recolección de URLs de hilos relevantes
2. Scraping individual de cada hilo
3. Limpieza y extracción de texto
4. Análisis con IA para identificar ofertas
5. Normalización de datos
6. Cálculo de diferencias
7. Eliminación de duplicados
8. Exportación a CSV/Excel
9. Generación de estadísticas
