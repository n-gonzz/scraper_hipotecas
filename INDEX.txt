🚀 COMANDOS RÁPIDOS
================================================================================

1. INSTALACIÓN
   cd /app/mortgage_scraper
   pip install -r requirements.txt

2. TEST DE VALIDACIÓN
   python test_scraper.py
   → Verifica que todo esté configurado correctamente

3. DEMO (3 hilos por foro)
   python demo.py
   → Scraping rápido para probar

4. SCRAPING COMPLETO (20 hilos por foro)
   python scraper.py
   → Extracción completa de ofertas

5. SCRAPING CON CONFIGURACIÓN PERSONALIZADA
   python run_with_config.py
   → Usa configuración de config.py

6. ANÁLISIS DE RESULTADOS
   python analyze.py
   → Genera estadísticas de los datos extraídos

🧠 FUNCIONALIDADES PRINCIPALES

   • Bogleheads España (https://bogleheads.es/foro/)
   • Forocoches (https://www.forocoches.com/)

   • OpenAI GPT-4o con Emergent LLM Key
   • Identificación automática de ofertas hipotecarias
   • Estructuración de datos no estructurados

✅ Filtrado inteligente:
   • Por keywords relevantes
   • Por fecha (últimos 24 meses por defecto)
   • Eliminación de duplicados

✅ Datos extraídos por oferta:
   • Banco
   • TIN Oficial
   • TIN Negociado
   • TAE
   • Diferencia (puntos porcentuales)

   • Resumen del caso
   • URL del hilo original

✅ Exportación múltiple:
   • CSV (compatible con Excel, Google Sheets)
   • Excel (.xlsx) con formato
   • JSON (opcional, configurable)

✅ Análisis de datos:
   • Estadísticas generales
   • Top bancos por número de ofertas
   • Mejores ofertas por TIN
   • Distribución por tipo de hipoteca
   • Análisis de condiciones comunes

================================================================================
⚙️ CONFIGURACIÓN PERSONALIZADA (config.py)
================================================================================

Parámetros modificables:

• MAX_THREADS_PER_FORUM = 20        # Hilos a procesar por foro
• MAX_POST_AGE_MONTHS = 24          # Antigüedad máxima de posts
• REQUEST_DELAY_SECONDS = 1         # Pausa entre requests
• AI_MODEL = "gpt-4o"               # Modelo de IA
• ENABLE_RANKIA = True              # Activar/desactivar foros
• EXPORT_CSV = True                 # Formatos de exportación
• LOG_LEVEL = "INFO"                # Nivel de logging

================================================================================
📈 FLUJO DE TRABAJO DEL SCRAPER
================================================================================

1. RECOLECCIÓN
   └─ Busca hilos relevantes en cada foro activo
   └─ Filtra por keywords y fecha

2. EXTRACCIÓN
   └─ Accede a cada hilo
   └─ Limpia HTML y extrae texto plano
   └─ Verifica relevancia del contenido

3. ANÁLISIS CON IA
   └─ Envía texto a GPT-4o
   └─ Extrae datos estructurados
   └─ Identifica ofertas reales con números

4. NORMALIZACIÓN
   └─ Estandariza formatos (porcentajes, fechas)
   └─ Calcula diferencias entre TIN oficial/negociado
   └─ Elimina duplicados

5. EXPORTACIÓN
   └─ Genera CSV con todos los datos
   └─ Crea Excel formateado
   └─ Muestra estadísticas en consola

================================================================================
🛡️ CONSIDERACIONES ÉTICAS Y LEGALES
================================================================================

✅ Datos públicos: Solo extrae información pública de foros
✅ No datos personales: No recopila emails, nombres, teléfonos
✅ Rate limiting: Incluye pausas para no sobrecargar servidores
✅ User-Agent: Identifica requests apropiadamente
✅ Robots.txt: Respeta políticas de scraping de cada sitio
✅ Uso educativo: Para investigación y comparación personal

================================================================================
🐛 SOLUCIÓN DE PROBLEMAS COMUNES
================================================================================

PROBLEMA: Module not found
SOLUCIÓN: pip install -r requirements.txt --extra-index-url https://d33sy5i8bnduwe.cloudfront.net/simple/

PROBLEMA: EMERGENT_LLM_KEY no encontrada
SOLUCIÓN: Verificar que existe /app/mortgage_scraper/.env con la key

PROBLEMA: Timeout en scraping
SOLUCIÓN: Reducir MAX_THREADS_PER_FORUM en config.py a 10 o menos

PROBLEMA: No se encuentran ofertas
SOLUCIÓN: Aumentar MAX_THREADS_PER_FORUM a 50+ para más datos

PROBLEMA: Rate limit de IA
SOLUCIÓN: Aumentar REQUEST_DELAY_SECONDS en config.py

PROBLEMA: Foro no accesible
SOLUCIÓN: Desactivar ese foro en config.py (ENABLE_NOMBRE = False)
